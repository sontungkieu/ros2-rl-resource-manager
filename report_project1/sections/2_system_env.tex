\section{System Environment}
The system was constructed with the following specific configuration:
\begin{itemize}
    \item \textbf{Host Platform:} Windows 11 with WSL 2.
    \item \textbf{Guest Operating System:} Ubuntu 24.04 LTS (Noble Numbat).
    \item \textbf{Middleware:} ROS 2 Jazzy Jalisco.
    \item \textbf{Simulator:} Gazebo Harmonic.
    \item \textbf{Robot Platform:} TurtleBot3 (Model: Burger).
\end{itemize}
